# assets/

Non-code resources used by the experiment at runtime.

| Path | Purpose |
|---|---|
| `images/1.png` -- `images/12.png` | Countdown timer images shown during trials and rest periods |
| `body_loc_instruction_vid_3T.mp4` | Participant instruction video for the 3T scanner |

If you change the number of countdown images, update `NUM_COUNTDOWN_IMAGES` in `src/parameters.py` to match.
